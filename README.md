## PART 1: LINUX ADMINISTRATION — THE FOUNDATION OF DEVOPS

### Author's Note to the Reader

Before we touch AWS, Docker, Kubernetes, or Jenkins, we need to master Linux. Here's why this matters for you specifically: every EC2 instance you launch, every Docker container you build, every Jenkins server you configure — all of it runs on Linux. Interviewers ask Linux questions first because if you can't navigate a terminal, nothing else matters. This part is long on purpose. Read it slowly, type every command yourself, and don't skip the "Common Mistakes" sections — they will save you hours of frustration.
